# Lazarus: Conectando Agricultores con Hostelería

## Descripción General
Lazarus es una plataforma web que conecta agricultores locales con restaurantes y hostelería, permitiendo la venta directa de productos ecológicos y sostenibles a precios justos, eliminando intermediarios y apoyando la economía local.

## Estructura del Proyecto

- `index.html`: Página principal de la plataforma, contiene la estructura y contenido de la landing page.
- `css/styles.css`: Hojas de estilo para el diseño visual y la experiencia de usuario.
- `js/main.js`: Lógica de interacción, validación de formularios, animaciones y utilidades.
- `img/`: Imágenes utilizadas en la web (productos, logotipos, etc).
- `docs/`: Carpeta destinada a documentación adicional (actualmente vacía).

## Funcionalidades Principales

- **Landing Page Responsive**: Diseño adaptable a dispositivos móviles y escritorio.
- **Navegación intuitiva**: Menú de navegación móvil y de escritorio.
- **Sección de productos**: Presenta los valores diferenciales (ecológico, directo, justo, conexión directa).
- **Galería de confianza**: Muestra productos y testimonios visuales.
- **Formulario de registro**: Permite a agricultores y restaurantes registrar su interés.
- **Animaciones y efectos visuales**: Animaciones al hacer scroll, feedback de formularios, mensajes emergentes.
- **Validación y almacenamiento local**: Validación de datos y almacenamiento temporal en localStorage.

## Instalación y Uso

1. Clona o descarga el repositorio.
2. Abre `index.html` en tu navegador preferido.
3. Personaliza imágenes y textos según tus necesidades.

## Tecnologías Utilizadas
- HTML5
- CSS3 (Flexbox, Grid, variables CSS)
- JavaScript (ES6+)
- Fuentes de Google Fonts (Poppins)
